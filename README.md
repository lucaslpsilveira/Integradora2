# Integradora 2

## Configurando o projeto

**Clone o repositório do git **

```sh
    git clone 
```

**Instale o npm**
```sh
    npm install -g npm
```

**Entre na pasta do projeto**
```sh
    cd integradora2
```

**Instale os requisitos do projeto**

```sh
    npm install
```

## Iniciando servidor de Desenvolvimento 

```sh
    npm start
```

## Build do projeto

```sh
    npm run build
```

## Rode a Build de Produção

```sh
    serve -s build
```
